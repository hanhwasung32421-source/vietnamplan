# vietnamplan

## 바로 열기

- GitHub Pages(설정 시): `https://<계정>.github.io/vietnamplan/`
- 로컬: `index.html`을 브라우저로 열면 자동으로 `다낭_혼숙호텔_및_관광정보.html`로 이동합니다.

## 배포(GitHub Pages) 설정

GitHub 저장소 설정에서 아래처럼 지정하면 커밋/푸시 후 바로 반영됩니다.

- Settings → Pages
- Source: `Deploy from a branch`
- Branch: `main` / `(root)`

## 자동 푸시

수정하면 자동 커밋/푸시가 필요하면 `auto_push.ps1`를 실행하세요.
